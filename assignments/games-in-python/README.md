

# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman game in Python. Practice string manipulation, loops, conditionals, and random selection by creating an interactive word-guessing game.

## 📝 Tasks

### 🛠️  Build the Hangman Game

#### Description
Create a Python program that lets a player guess letters to reveal a hidden word before running out of attempts. The game should randomly select a word from a list and provide feedback after each guess.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses from the user
- Display the current progress (e.g., _ _ t h o n)
- Track and display the number of incorrect guesses remaining
- End the game when the word is fully guessed or attempts are exhausted
- Show a win or lose message at the end

#### Example
```
Word: _ _ _ _ _ _
Guess a letter: p
Word: p _ _ _ _ _
Incorrect guesses left: 5
Guess a letter: z
Word: p _ _ _ _ _
Incorrect guesses left: 4
...
```
